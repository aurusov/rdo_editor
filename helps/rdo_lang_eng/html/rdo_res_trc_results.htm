INDEXES TRACING

An index tracing line has the following format:

V <time> <index_number> <value>
time
Field "time" contains the moment the index value changes written in real constant format.
index_number
The index number is the number indicated for the indexes traced in section $Watching of the tracing object.
value
Indexes are traced at the moment of changing the parameter values, the quantity of temporary resources, the logical expression value (for indexes of watch_state kind), or at the moment the resource is eliminated (for indexes of watch_value kind).

The value depends on the index kind. Values are given below for indexes of every kind:

Index kind Value description
watch_par Parameter's new value
watch_state Logical expression's new value
watch_quant Resource quantity new value
watch_value Arithmetic expression's value

For indexes of get_value kind there is no tracing.
