CERTAIN CONSTRUCTIONS OF THE LANGUAGE

SM on RAO language consists of the objects.

There is a minimum set of objects, which is obligatory for any model. The rest of the objects can be added to the obligatory set by user or can be created by the simulator.

The minimun set of objects of SM on RAO includes: resource types, patterns, resources, operations (or decision points), run.

In addition to these obligatory objects may be created and used following input data objects: constants, functions and sequences, animation frames, performance measures (statistics).

The object of resource types and the object of resources are described in the previous issue. In this issue the description of the rest objects of SM is given.
