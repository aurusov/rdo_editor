RESOURCES STATE TRACING

The tracing of resources states includes information about all the changes of resources states. At the moment a traced resource's state changes, a separate line is put into the tracing object with the following format:

R<status_symbol> <time> <resource_type_number> <resource_number>
<new_values_of_parameters>
status_symbol
Status symbol shows what has happened with the resource as a whole:
Status symbol Description
C The resource has been created
E The resource has been eliminated
K The resource parameter values have changed
time
Field "time" contains the moment the resource state changes written in real constant format.
resource_type_number, resource_number
The resource type number and the resource number are determined by the order of types and resources in the type describing object and in the resource object correspondingly.
new_values_of_parameters
New parameter values are put into the object in the order of parameters in the resource types description. For enumerative parameters, not the value names are put out but their numbers. The first value name is assigned number 0, the second one is assigned number 1 and so forth.

For resources to be eliminated (status symbol E), new parameter values are not put out.
