STRING CONSTANTS

A string constant is a text line bound in quotation marks. The text may contain separators (except end of line and end of object signs) or symbols not belonging to the alphabet. No quotation marks are allowed within a string constant.

Remarks

The maximum length of a string constant, the bounding quotation marks included, is 63 symbols.

Examples

'Abc{def}j'
'1743+hdfs-'
'hhh kkk'
'Line ____'
