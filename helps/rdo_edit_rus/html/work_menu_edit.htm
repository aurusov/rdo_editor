МЕНЮ EDIT

Данное меню предоставляет функции для редактирования текста модели.

Пункт меню Комбинация клавиш Функция
Undo Ctrl+Z Последовательный откат изменений, внесенных в модель в процессе редактирования
Redo Ctrl+Y Последовательное воспроизведение изменений, внесенных в модель в процессе редактирования
Cut Ctrl+X Вырезание выделенного фрагмента текста модели в буфер обмена
Copy Ctrl+C Копирование выделенного фрагмента текста модели в буфер обмена
Paste Ctrl+V Вставка текста из буфера обмена
Delete Del Удаление выделенного фрагмента текста модели
Copy As RTF Копирование выделенного фрагмента текста модели в буфер обмена c сохранением форматирования и цветового выделения синтаксических конструкций языка
Select All Ctrl+A Выделение всего текста в текущем редактируемом объекте модели
Make Selection Uppercase Ctrl+Shift+U Изменение регистра в выделенном фрагменте текста на верхний, т.е. при выборе данного пункта меню выделенный текст будет записан прописными буквами
Make Selection Lowercase Ctrl+U Изменение регистра в выделенном фрагменте текста на нижний, т.е. при выборе данного пункта меню выделенный текст будет записан строчными буквами
Comment Selection Ctrl+Q Установка символов начала комментария слева и окончания комментария справа от выделенного фрагмента текста
Complete Word Ctrl+Space Включение механизма автоматического завершения синтаксических конструкций языка
