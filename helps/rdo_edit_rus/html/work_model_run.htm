ЗАПУСК, НАСТРОЙКА ЗАПУСКА

Для обеспечения возможности запуска моделей необходимо указать местоположение РДО-имитатора. Для этого необходимо выбрать пункт меню RAO->Path или нажать на копку панели инструментов проекта. Далее в открывшемся стандартном диалоге открытия файла WINDOWS необходимо указать путь к РДО-имитатору.

Запуск модели осуществляется путем выбора пункта меню RAO->Run, нажатием клавиши F5 или нажатием на кнопку в панели инструментов проекта. Если при попытке запуска модели возникает ошибка синтаксиса или ошибка в процессе прогона, то выдается сообщение об ошибке, а курсор устанавливается на позицию (строку и символ), в котором обнаружена ошибка. Информация о последней ошибке отражается в информационной панели строки состояния.

Остановить прогон, можно выбрав пункт меню RAO->Stop, нажатием комбинации клавиш Shift+F5 или нажатием на кнопку в панели инструментов проекта.

Существует возможность настройки взаимодействия программного комплекса с РДО-имитатором.
