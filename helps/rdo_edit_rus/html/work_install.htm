УСТАНОВКА ПРОГРАММНОГО КОМПЛЕКСА

Для обеспечения работы программного комплекса необходимо чтобы модули интерфейса пользователя и файлов справок находились в одной директории (папке) на диске.

Расположение РДО-имитатора может быть произвольным. Для обеспечения запуска редактируемых моделей необходимо в программном комплексе указать путь к РДО-имитатору.
