МЕНЮ Model

Данное меню предоставляет функции для работы с РДО-имитатором и результатами моделирования.

Пункт меню Комбинация клавиш Функция
Run F5 Запуск модели
Stop Shift+F5 Остановка прогона
Path   Указание пути к РДО-имитатору
Reload PMV file   Повторное считывание объекта результатов
Reload TRC file   Повторное считывание объекта трассировки
