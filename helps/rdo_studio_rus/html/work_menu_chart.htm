МЕНЮ CHART

Данное меню предоставляет функции для работы с графиками.

Пункт меню Комбинация клавиш Функция
Find In Charts   Переход к графику по его имени
Add To New Chart   Добавление нового графика
Time Wrap   Включает/отключает режим "переноса" времени для выбранного графика при его отображении
Options   Вызывает диалоговое меню, позволяющее менять свойства выбранного графика
