НАСТРОЙКИ

Изменение настроек программного комплекса возможно путем выбора пункта меню View->Options. При этом выводится диалоговое окно настроек, имеющее две закладки - Edit и Color.

Справочная информация по каждой из закладок доступна при выборе соответствующей закладки и нажатии кнопки Справка (Help).

Доступна также контекстная справка по каждому из элементов управления. Для этого необходимо нажать на кнопку в заголовке окна настроек, а затем кликнуть на интересующий элемент управления.

В группе Tabs настраиваются знаки табуляции и отступы:

Наименование настройки Назначение
Tab size Размер табуляции в символах пробела
Indent size Размер отступа в символах пробела
Use tabs Если не установлен этот флажок, то при нажатии на клавишу Tab на клавиатуре в текст модели вместо символа табуляции будут вставлены пробелы, причем их количество определяется значением Tab size
Tab indents Если установлен данный флажок, то нажатие на клавишу Tab на клавиатуре приведет к вставке в текст модели отступа размером Indent size
Backspace Untabs, Backspace Unindents Возможен выбор одного из вариантов. В случае, если выбран первый вариант, то при нажатии клавиши Backspace будет удален символ табуляции (если он присутствует). Если выбран второй вариант, то при нажатии на клавишу Backspace будет удален весь отступ целиком (может состоять из нескольких символов табуляции и пробелов)
Auto indent Включение механизма автоматического переноса отступа от предыдущей строки к следующей при переводе каретки

В группе Auto complete настраиваются параметры автозавершения синтаксических конструкций языка:

Наименование настройки Назначение
Use auto complete Включение/выключение режима автозавершения по нажатию комбинации клавиш Ctrl+Space
Show full list, Show nearest words only Возможен выбор только одного из вариантов. Если выбран первый вариант, то выпадающий список зарезервированных слов языка РДО будет содержать все ключевые слова, а курсор будет позиционирован на ближайшем слове, начало которого совпадает по написанию с символами, введенными пользователем перед нажатием Ctrl+Space. Если выбран второй вариант, то выпадающий список зарезервированных слов языка РДО будет содержать только те слова, начало которых совпадает по написанию с символами, введенными пользователем

В группе Debug настраиваются параметры и возможности по отладке моделей, включающие в себя запуск модели, остановку процесса имитации из программного комплекса, отслеживание момента окончания моделирования и обработку ошибок (см. Меню RAO). Данная группа введена для совместимости с различными версиями РДО-имитатора и различными операционными системами:

Наименование настройки Назначение
None Если выбран данный вариант, то пользователь имеет только возможность запуска модели. Момент окончания моделирования не отслеживается. Возможность остановки моделирования из программного комплекса отсутствует. Обработка ошибок не производится
Full Если выбран данный вариант, то пользователь имеет все возможности по отладке моделей
Run/Stop only Если выбран данный вариант, то пользователь имеет возможность запуска модели. Момент окончания моделирования отслеживается c помощью системного таймера. Имеется возможность остановки моделирования из программного комплекса. Обработка ошибок не производится
Run only Если выбран данный вариант, то пользователь имеет возможность запуска модели. После запуска модели программный комплекс ожидает окончания процесса моделирования (не возможно редактирование модели в процессе прогона). Обработка ошибок не производится. По окончании работы РДО-имитатора возможно дальнейшее редактирование модели

В группе Buffers настраиваются параметры автоматической очистки буферов:

Наименование настройки Назначение
Clear automatically Включение/выключение режима автоматической очистки буферов
Clear after Время в секундах от момента последнего добавления фрагмента в буфер до установки режима очистки перед следующим добавлением

В группе Window настраиваются функции переноса и горизонтальной прокрутки:

Наименование настройки Назначение
Word wrap Если данный флажок установлен, то включается режим переноса текста по словам
Horizontal scroll bar Если данный флажок не установлен, то горизонтальная полоса прокрутки не будет отображаться

Закладка Color

На данной закладке пользователь может настроить режим цветового выделения синтаксических конструкций языка.

Увеличить рисунок

Уменьшить рисунок

Theme - понятие темы включает в себя все настойки цветового выделения. Из выпадающего списка можно выбрать одну из подготовленных заранее тем. В нижней части закладки расположено окно предварительного просмотра темы. На базе любой из подготовленных заранее тем пользователь может разработать свою тему. Изменения отображаются в окне просмотра, расположенном в нижней части закладки.

Наименование шрифта и его размер являются едиными для всех элементов темы. Для изменения шрифта необходимо выбрать его наименование из выпадающего списка. Доступны только шрифты с фиксированной шириной символа.

В левой верхней части закладки приведен список элементов цветового выделения темы, для которых могут быть изменены параметры отображения. Для этого необходимо выбрать элемент из списка. Справа от списка находятся настройки, которые могут быть изменены для элемента:

Наименование настройки Назначение
Foreground, Background Настройка цвета элемента и его фона. Существует возможность выбрать цвет из выпадающего списка, а также задать собственный цвет, нажав на кнопку .... При этом откроется стандартный диалог WINDOWS для выбора цвета. Выбранный цвет добавляется в список и, если не присутствовал в списке изначально, отображается в виде тройки [R, G, B]
Bold, Italic, Underline С помощью этих флажков задаются параметры шрифта темы для выбранного элемента

Из выпадающего списка Bookmark пользователь может выбрать форму маркера закладки.

Из выпадающего списка Fold пользователь может выбрать форму маркера сворачиваемых фрагментов текста, в том числе возможно отображение границ сворачиваемых фрагментов.
