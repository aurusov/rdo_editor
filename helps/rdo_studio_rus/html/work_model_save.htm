СОХРАНЕНИЕ МОДЕЛИ

Сохранить модель можно путем выбора пункта меню File->Save, нажав комбинацию клавиш Ctrl+S или путем нажатия соответствующей кнопки на панели инструментов проекта - .

Если модель создавалась и не была сохранена, то выдается стандартное диалоговое окно WINDOWS для сохранения файла, где пользователю предлагается ввести имя файлов модели (файла прогона с расширением *.smr).

Существует возможность сохранения модели под другим именем. Для этого необходимо выбрать пункт меню File->Save As. При этом будет выдано стандартное диалоговое окно WINDOWS для сохранения файла, где пользователю предлагается ввести новое имя файлов модели (файла прогона с расширением *.smr).
