ПРОСМОТР РЕЗУЛЬТАТОВ МОДЕЛИРОВАНИЯ

Результатами моделирования являются объекты:

  • объект результатов, который выводится на закладку Results окна вывода автоматически при завершении прогона
  • объект трассировки, который отображается в режиме реального времени в процессе моделирования на закладке Trace окна вывода.
