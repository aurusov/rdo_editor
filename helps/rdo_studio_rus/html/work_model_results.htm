ПРОСМОТР РЕЗУЛЬТАТОВ МОДЕЛИРОВАНИЯ

Результатами моделирования являются объекты результатов (имеет расширение *.pmv и отображается на закладке PMV главного окна) и объект трассировки (расширение - *.trc, закладка - TRC).

При каждом новом прогоне возможно изменение данных объектов, т.к. они генерируются РДО-имитатором. Автоматическое обновление содержимого полей редактирования на закладках PMV и TRC не производится из-за возможного большого объема объектов. Пользователь имеет возможность самостоятельно принудительно обновить содержимое полей ввода на этих закладках.

Для обновления объекта результатов необходимо выбрать пункт меню RAO->Reload PMV file или нажать на кнопку в панели инструментов проекта.

Для обновления объекта трассировки необходимо выбрать пункт меню RAO->Reload TRC file или нажать на кнопку в панели инструментов проекта.
