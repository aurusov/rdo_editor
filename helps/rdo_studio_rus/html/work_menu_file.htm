МЕНЮ FILE

С помощью пунктов меню File осуществляются следующие действия с моделью:

Пункт меню Комбинация клавиш Функция
New Ctrl+N Создание новой модели
Open Ctrl+O Открытие существующей модели
Reopen Повторное открытие ранее открывавшейся модели
Save Ctrl+S Запись изменений в модели
Save As Запись модели под другим именем
Save All Запись всех открытых документов
Exit Alt+F4 Выход из программного комплекса
