МЕНЮ SEARCH

Данное меню предоставляет функции для поиска в тексте модели и навигации по модели.

Пункт меню Комбинация клавиш Функция
Find Ctrl+F Вызов стандартного диалога WINDOWS для поиска подстроки в тексте
Find Next F3 Повторный поиск найденного фрагмента в направлении к концу текста
Find Previous Shift+F3 Повторный поиск найденного фрагмента в направлении к началу текста
Replace Ctrl+R Вызов стандартного диалога WINDOWS для замены подстроки в тексте
Find in Model Вызов стандартного диалога WINDOWS для поиска подстроки в тексте всей модели. Результаты выводятся на закладку Find in Model окна Output
Toggle Bookmark Ctrl+F2 Установка/снятие закладки в тексте
Next Bookmark F2 Переход к следующей закладке
Previous Bookmark Shift+F2 Переход к предыдущей закладке
Clear Bookmarks Ctrl+Shift+F2 Удаление всех установленных в модели закладок
