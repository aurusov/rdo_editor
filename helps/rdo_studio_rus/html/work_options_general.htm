В группе File Association настраивается привязка системы моделирования к расширению файла прогона модели:

Наименование настройки Назначение
Setup file association for project file (*.smr) Автоматическая установка привязки к файлу прогона
Check association in future Проверять наличие привязки расширения при каждом запуске

Настройки, не вошедшие в группы:

Наименование настройки Назначение
Open last project at startup Автоматически загружать последнюю модель при запуске системы
Show project's full name on application's caption Выводить полнуть путь к модели в заголовке окна
