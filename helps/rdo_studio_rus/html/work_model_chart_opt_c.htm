СВОЙСТВА ОБЛАСТИ ОТОБРАЖЕНИЯ

В этой вкладке перечислены свойства, относящиеся к области отображения графика.

  • Title – Заголовок графика.
  • Values count on X axis, Values count on Y axis – максимальное количество подписей данных на осях абсцисс и ординат, соответственно.
  • Draw legend – Отображать соответствие формы графика к названию параметра.
  • Y axis values – Если на одной области построено несколько графиков, то этим параметром определяется значения какого из выбранных ресурсов будут отображены на оси ординат.
